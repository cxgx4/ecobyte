# EcoByte 🌍

> **Navigating the Future of Urban Ecology**

EcoByte is a futuristic, AI-driven environmental monitoring and safe-routing web application designed to help citizens navigate polluted urban landscapes safely. Built with a sleek, glowing neon-green cyberpunk aesthetic, EcoByte provides real-time, hyper-local ecological intelligence.

## ✨ Key Features

- 🔋 **Unified Command Dashboard:** A high-tech operations center displaying live environmental metrics (AQI, Temperature, Humidity) and predictive trends using dynamic charts.
- 🗺️ **AuraPath Matrix:** Our proprietary safe-routing engine. Instead of just finding the *fastest* route, AuraPath finds the *cleanest* route by analyzing OpenRouteService API data against environmental factors. 
  - 🟩 **Neon Green Routes:** The "Cleanest" path (Highest Health Score).
  - 🟦 **Solid Blue Routes:** The "Fastest" path.
- 🚨 **Real-Time Alerts:** Proactive warnings for sudden ecological changes, smog spikes, and extreme weather conditions.
- 🖧 **System Status Engine:** Real-time infrastructure monitoring, API connection latency tracking, and sensor health visualization presented in a "server room" aesthetic.
- 🌙 **Immersive Dark Mode UI:** Premium, glassmorphic interfaces with smooth micro-animations built using Framer Motion.

## 🛠️ Technology Stack

- **Frontend Core:** React 19, Vite, Tailwind CSS
- **Mapping & GIS:** MapLibre GL JS, React-Map-GL, Turf.js
- **Routing & APIs:** OpenRouteService
- **Data Visualization:** Recharts
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 🚀 Getting Started

Follow these steps to run EcoByte locally on your machine.

### Prerequisites
Make sure you have Node.js and npm installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/cxgx4/ecobyte.git
   cd ecobyte
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/` to access the EcoByte neural net.

## 📜 Scripts

- `npm run dev`: Starts the local Vite development server.
- `npm run build`: Compiles the application for production.
- `npm run lint`: Runs ESLint to check for code quality issues.
- `npm run preview`: Previews the built production application locally.
